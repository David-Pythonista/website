QR code detector

Description of program:

Select an input image with one or more QR code(s) and the program will find them.

First section is responsible for importing the modules needed for the project. The cv2 is an image processing library with useful image handling properties. Tkinter is for making the GUI, and PIL is for handling and converting the different formats of images. In this case, we have to use the square root function from the math module
The init method will create the user interface, with a blank canvas, and two control buttons.

When the user clicks on the start button, it will call the main functions of the program. In the first step, determine all contours on the input image, and if there are any QR codes, draw their contours. The last function will display it.

The load button provides the opportunity to choose an image file for processing. It is not just making the connection between the operating system, also collects the contours of the image. After this call the show_image function.

It converts the image to make it possible to insert and display it on the canvas what is created by the init function. For this purpose, use resize and interpolation.

Calculate the area of the given shape, which is referred to as the number of the contour. Calculate the interval between two given points and return it.

The list_of_finding_patterns function will determine the contours that are QR codes. The finding patterns (the black rectangles in the three corners) have a specific size ratio. To use this, it can decide if the contour belongs to a QR code finding pattern or not. These rectangles are used to find the 2D matrixes, they have a specified distance from each other. That's why we need to determine_area and, in the next section, the distance function.

The "draw_the_QR_code", will check the coordinates and find the real QR codes in the image. It has to use three loops to go through the lists of coordinates.

End of description.
